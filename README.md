## Who We Are

**GRANDOU Agency** is a digital agency founded to transform the online presence of businesses in Haiti and the diaspora. We craft identities that leave a mark, web experiences that engage, and growth systems that last.

We don't just build websites — we build **digital assets**.

---

## What We Do

```
  Identity              Web Experiences          Systems
─────────────       ─────────────────────    ──────────────────
  Branding              SPA / Landing            Automation
  Logo Design           E-commerce               CRM / Pipelines
  UI / UX               Client Portals           SEO / Analytics
  Design System         Full Redesigns           Email Marketing
```
